## API Reference

### Pagination

The root navigation container. Renders a `<nav>` with `aria-label="Pagination"`. Supports keyboard navigation with Arrow Left/Right and Home/End keys.

### PaginationItem

<PropsTable data={[
  { name: "isActive", type: "boolean", defaultValue: "-", description: 'Marks this item as the current page. Adds aria-current="page".' }
]} />
