### Fallback

Displays initials when no image is available.

<ComponentPreview codeFile="avatar/example-3.txt">
  <Avatar>
      <AvatarFallback>JD</AvatarFallback>
  </Avatar>
</ComponentPreview>

### Group

Display multiple avatars with an overflow indicator.

<ComponentPreview codeFile="avatar/example-4.txt">
  <AvatarGroup max={3}>
      <Avatar><AvatarFallback>A</AvatarFallback></Avatar>
      <Avatar><AvatarFallback>B</AvatarFallback></Avatar>
      <Avatar><AvatarFallback>C</AvatarFallback></Avatar>
      <Avatar><AvatarFallback>D</AvatarFallback></Avatar>
      <Avatar><AvatarFallback>E</AvatarFallback></Avatar>
  </AvatarGroup>
</ComponentPreview>

## API Reference

<PropsTable>
  <PropsRow name="max" type="number" defaultValue="3">
