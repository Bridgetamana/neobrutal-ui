## API Reference

<PropsTable>
  <PropsRow name="variant" type="&quot;default&quot; | &quot;noShadow&quot; | &quot;neutral&quot;" defaultValue="&quot;default&quot;">

  </PropsRow>
  <PropsRow name="size" type="&quot;default&quot; | &quot;sm&quot; | &quot;lg&quot; | &quot;icon&quot;" defaultValue="&quot;default&quot;">
    The size of the button.
  </PropsRow>
  <PropsRow name="asChild" type="boolean" defaultValue="false">
    Render as a child element (e.g., Link) using the Slot pattern.
  </PropsRow>
</PropsTable>
