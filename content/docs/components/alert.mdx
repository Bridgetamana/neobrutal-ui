## API Reference

### Alert

<PropsTable data={[
  { name: "variant", type: '"default" | "neutral" | "destructive" | "success" | "warning"', defaultValue: '"default"', description: "Controls the color scheme and default icon of the alert." },
  { name: "icon", type: "React.ReactNode", defaultValue: "-", description: "Custom icon to display. Pass null to hide the icon entirely." },
]} />

`AlertTitle` and `AlertDescription` extend the native `<h5>` and `<div>` elements respectively and accept all standard HTML attributes.
