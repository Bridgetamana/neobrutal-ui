## Examples

### Types

Four built-in types, each with its own color.

<ComponentPreview code="toast.success('Operation completed')\ntoast.error('Something went wrong')\ntoast.warning('Disk space is running low')\ntoast.info('A new version is available')">
  <div className="flex w-full justify-center">
    <ToastTypesDemo />
  </div>
</ComponentPreview>

### Action button

Pass <code className="font-mono text-xs bg-white px-1 py-0.5 border-2 border-black rounded-base font-bold shadow-brutal">actionProps</code> to render a neobrutalist action button inside the toast.

<ComponentPreview code="toast.success('File deleted', {\n  title: 'Deleted',\n  actionProps: {\n    children: 'Undo',\n    onClick: () => toast.info('File restored'),\n  },\n})">
  <div className="flex w-full justify-center">
    <ToastActionDemo />
  </div>
</ComponentPreview>

### Promise

Show a loading state that automatically transitions to success or error when the promise settles.

<ComponentPreview code={`toast.promise(saveDocument(), {\n  loading: "Saving document...",\n  success: (data) => ({\n    title: "Saved",\n    description: \`\${data.name} saved successfully\`,\n  }),\n  error: (err) =>\n    err instanceof Error\n      ? err.message\n      : "Something went wrong",\n})`}>
  <div className="flex w-full justify-center">
    <ToastPromiseDemo />
  </div>
</ComponentPreview>

### Position

Set the toast position by passing a <code className="font-mono text-xs bg-white px-1 py-0.5 border-2 border-black rounded-base font-bold shadow-brutal">position</code> prop to the Toaster component in your root layout.

<CodeBlock code="import { Toaster } from '@/components/ui/toast'\n\n// In your root layout\n<Toaster position='top-right' />" />
