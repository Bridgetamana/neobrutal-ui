## Using the CLI

This guide will walk you through the installation steps for using Neobrutal UI via the CLI.

### 1. Initialize your project

<CodeBlock code="npx neobrutal init" language="bash" />

### 2. Add a component

<CodeBlock code="npx neobrutal add button" language="bash" />

### 3. Import and use

<CodeBlock code="import { Button } from '@/components/ui/button'\n\nexport default function Page() {\n  return <Button>Click me</Button>\n}" />

## Manual Installation

Follow these steps to add components to your project without the CLI

### 1. Install dependencies

<CodeBlock code="npm install class-variance-authority clsx tailwind-merge" language="bash" />

### 2. Create the utility function

Add this to `lib/utils.ts`

<CodeBlock code="import { clsx, type ClassValue } from 'clsx'\nimport { twMerge } from 'tailwind-merge'\n\nexport function cn(...inputs: ClassValue[]) {\n  return twMerge(clsx(inputs))\n}" />

### 3. Copy the component

Browse to any component page, open the Code tab, and copy the source into `components/ui/`. Install Base UI if required.

## HTML Only

Don't use React? Every component includes a plain HTML version with Tailwind classes. Open any component page, select the Code tab, switch to HTML, and paste it into the [Tailwind CSS Playground](https://play.tailwindcss.com/) to try it out.
