title: Accessibility
description: Learn about how Neobrutal UI is built with accessibility in mind.
---

Components use semantic HTML and are built on [Base UI](https://base-ui.com) for keyboard navigation, focus management, and screen reader support.

## Focus Indicators

All interactive components have visible focus states that meet contrast requirements. Modal dialogs trap focus to prevent tabbing outside the dialog.

## Motion

Animations respect `prefers-reduced-motion`. Users who prefer reduced motion will see minimal or no animations.
