---
title: Theming
---

Edit the CSS variables in your **globals.css** to customize colors, shadows, and border radius.

```css
:root {
  --main: #B6ACE4;   
  --bg: #f0eefc;    
  --white: #ffffff;  
  --black: #000000;   
  --radius: 5px;     
  --shadow-brutal: 4px 4px 0px 0px var(--black);
}
```
